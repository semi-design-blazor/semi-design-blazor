## Semi Design Blazor？

基于`semi-design`设计规范和`BlazorComponent`的交互能力提供标准的基础组件库。 提供如布局、弹框标准、Loading、全局异常处理等标准场景的预置组件。从更多实际场景出发，满足更多用户和场景的需求，最大的减少开发者的时间成本。缩短开发周期提高开发效率。并提供一套Web解决方案示例 有多种常见场景和预设布局等精彩内容。
